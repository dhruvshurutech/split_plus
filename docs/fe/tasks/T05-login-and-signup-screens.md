# T05 Login and Signup Screens

## Depends On
- T01
- T03

## Goal
Implement authentication entry screens.

## Scope
- Build `/login` and `/signup` forms.
- Wire APIs: `POST /auth/login`, `POST /users/` then login.
- Handle validation and error display.

## Deliverables
- Functional login screen.
- Functional signup screen.
- Redirect into app after success.

## Acceptance Criteria
- Valid credentials log user in.
- Signup creates account and enters app.
- Error states shown with actionable copy.
