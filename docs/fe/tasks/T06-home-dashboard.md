# T06 Home Dashboard

## Depends On
- T02
- T04

## Goal
Implement authenticated home summary screen.

## Scope
- Route: `/(app)/`.
- Load `GET /users/me`, `GET /users/me/balances`, `GET /groups/`.
- Show concise mobile cards and quick actions.

## Deliverables
- Dashboard layout and loaders.
- Summary widgets for profile, balances, and groups.

## Acceptance Criteria
- Screen loads all three data sets.
- Loading, empty, and error states present.
