# T11 Expense List and Detail

## Depends On
- T02
- T08

## Goal
Implement expense list and expense detail screens.

## Scope
- Group tab list from `GET /groups/{group_id}/expenses/`.
- Detail route with `GET /groups/{group_id}/expenses/{expense_id}/`.
- Update and delete actions.

## Deliverables
- Expense list UI.
- Expense detail UI.
- Update/delete flows.

## Acceptance Criteria
- Expense list navigates to detail.
- Update and delete mutate data correctly.
