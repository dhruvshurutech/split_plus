# T09 Group Members and Invitations

## Depends On
- T02
- T08

## Goal
Implement member list and invite flow inside group detail.

## Scope
- Load `GET /groups/{group_id}/members`.
- Invite via `POST /groups/{group_id}/invitations`.
- Display invite success token/message from response.

## Deliverables
- Members list section.
- Invite member form.

## Acceptance Criteria
- Members render correctly.
- Invite request succeeds and feedback is shown.
