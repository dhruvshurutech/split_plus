# T07 Groups List and Create Group

## Depends On
- T02
- T04

## Goal
Build groups index and create flow.

## Scope
- Route: `/(app)/groups`.
- Load `GET /groups/`.
- Create group via `POST /groups/`.

## Deliverables
- Group list UI.
- Mobile create group sheet/modal.
- Successful create refreshes list.

## Acceptance Criteria
- New group appears without full reload.
- Validation and API errors are visible.
